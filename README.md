## 🤔 什么是黑U？

- **标记U**：已被安全机构或交易所标记、冻结的USDT，通常涉及非法活动（如洗钱、诈骗）。

1. **小额返利**：初期充值小额USDT（如50-100U），立即按承诺返利以建立信任。
2. **大额收割**：待用户充值大额后，以“IP异常”、“需激活通道”等借口拒绝返款，甚至要求继续充值。
